ChangeAccountability Settings

How to get a GitHub Personal Access Token:

  1. Go to GitHub Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Click "Generate new token"
  3. Give it a descriptive name like "ChangeAccountability Extension"
  4. Set expiration as needed (90 days recommended)
  5. For Repository access, select "Public Repositories (read-only)" or specific repositories
  6. Under Permissions, grant: Contents: Read, Metadata: Read, Pull requests: Read
  7. Click "Generate token" and copy the token
This token is stored locally and used to fetch PR metadata from GitHub's API. It's only sent to github.com and api.github.com.
